# Compilação e Execução

## Compilação
Para compilar, use o seguinte programa:
```sh
 gcc main.c -o quebra_senhas -pthread
```
## Execução
 Para executar o programa, use o seguinte comando:

### Para executar processos: 
```sh
./quebra_senhas -p
```

### Para executar Threads:
```sh
./quebra_senhas -t
```
